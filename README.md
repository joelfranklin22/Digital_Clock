# Digital Clock ⏰

A modern and responsive Digital Clock web application built using HTML, CSS, and JavaScript.

## 🚀 Features

- Live Real-Time Clock
- 24 Hour Digital Format
- Responsive UI Design
- Multiple Clock Templates
  - Digital
  - Digital Blue
  - Digital Red
  - Digital Orange
- Dark Theme UI
- Animated Toggle Icons
- Fullscreen Icon Support
- Modern Settings Panel
- Interactive Template Selection

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Font Awesome Icons
