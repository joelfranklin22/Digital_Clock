# 🕐 Digital Clock

A clean and minimal digital clock web app built with HTML, CSS, and JavaScript.

## Features

- Live digital clock with real-time updates
- Stopwatch / Timer with start, pause, and reset
- Clock / Timer mode switch
- Theme switcher — Default, Blue, Red, Orange
- Light / Dark mode toggle
- Date and Day display toggle
- Fullscreen support
- Fully responsive — mobile and tablet friendly
- Custom digital font (Quartz)

## Tech Stack

- HTML
- CSS
- JavaScript (Vanilla)
- Font Awesome Icons
